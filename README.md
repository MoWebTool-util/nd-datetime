# nd-datetime

[![spm version](http://spmjs.io/badge/nd-datetime)](http://spmjs.io/package/nd-datetime)

> 简单的时间日期转换库

## 安装

```bash
$ spm install nd-datetime --save
```

## 使用

```js
var Datetime = require('nd-datetime');
// use Datetime
```
## 开发

### 本地 Web 服务

```bash
grunt
```

浏览器中访问 http://127.0.0.1:8851

### 生成/查看 API 文档

```bash
grunt doc
grunt
```

浏览器中访问 http://127.0.0.1:8851/doc

### 代码检查与单元测试

```bash
grunt test
```

### 发布组件到 SPM 源

```bash
grunt publish
```
